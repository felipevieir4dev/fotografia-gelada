# 📸 Fotografia Gelada

Galeria de imagens, feita em React.


## 🛠️ Como Rodar

1. Clone o repositório:
   ```bash
   git clone https://github.com/seu-usuario/fotografia-gelada.git
   ```

2. Entre na pasta:
   ```bash
   cd fotografia-gelada
   ```

3. Instale as dependências:
   ```bash
   npm install
   ```

4. Rode o projeto:
   ```bash
   npm start
   ```

Abra no navegador: `http://localhost:3000`
